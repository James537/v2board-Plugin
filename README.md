## V2Ray-API

### 安装

1. 下载

   ```bash
   git clone https://github.com/thank243/V2Board_Plugin.git
   ```

2. 安装依赖

   ```bash
   pip3 install grpcio google-api-python-client
   ```


### 使用

   ```bash
   chmod +x main.py
   chmod +x v2ray -R
   ```
   
   修改cfg.json中的内容
   
   ```bash
   ./main.py
   ```
